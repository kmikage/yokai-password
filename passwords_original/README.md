# 解析されたパスワード

このディレクトリに、解析されたパスワードを収集していきます。  
解析したパスワードを共有された際、 Discordチャンネル内の「パスワード掲載用」でお知らせ頂けると幸いです。

## ディレクトリ名
投稿された方のお名前  
(ファイルシステムの都合上、一部変更する場合があります)

## ファイル形式
- README.txt  
更新履歴の管理。
以下の情報を随時追加していきます。  
  - 投稿時間お名前
  - コメント
  - originのURL

- passwords_*.txt  
originよりダウンロードしたパスワード
ファイル形式：1ファイル1パスワード

## 共通事項
- 文字コードはUTF-8
